# Let them cook

This application is a recipe app where users can look at recipes, check their ingredients, add new ones and save their favourites!

## Recipe Search 

The recipe search allows users to search for any recipe they want. Searching can be done by entering a keyword into the field and pressing the search button, or by using the category/area dropdowns to search by ingredient or area.   

Recipes can be viewed by clicking the view recipe button underneath as you scroll down the window.

### Random Recipe 
In the event that a user wants to try something new and is unsure of what to choose, the user can click the random recipe button to generate a random recipe.

## Favourite Recipes

When view recipe is clicked, the user gets a pop up of the recipe page. At the bottom of this page is the favourite button. When the favourite button is clicked, the recipe view updates to show that the star next to the favourite button is filled it. If the user clicks out of the recipe view and clicks the favourite recipies button, they can see a list of all the recipes they have favourited. 

## Ingredient View 
The user can click ingredient view to see a list of all the ingredients they have. They can also choose to add or remove ingredients by entering text into either box and clicking the corresponding button. The add ingredient button will update the view with the new ingredient, while the remove ingredient button will update the view to no0 longer have the ingredient. If the ingredient is not there, then an error message will pop up. 

Ingredients are updated in the user account information, so all changes are recorded to the file storing user information.

## Add new recipe 
The user can add a new recipe by clicking the add recipe button, and creating a list of instructions and ingredients to create the recipe using the prompts listed. Once they click submit, the user can see the recipes they have by clicking the my recipes button and seeing the recipes included.
